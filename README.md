# Curso Alura Git

### Instalação

Site Oficial: 

* https://git-scm.com 


### Comandos Básicos

##
Para conferir a instalação:

    git --version
    
##  
Para iniciar um repositório, dentro  do diretório desejado:

    git init
    
## 
Para saber informações sobre o repositório atual:
    
    git status
